template<typename ContextType>
void Zeta::ContextOperation< ContextType >::notifyWaiters ( )
inline

Notifies all waiters that called waitToBeCompleted() to continue. This in most cases is called by a context. Calling it with the Handled flag=false will literally do nothing, except consume system resources. So don't call it from anywhere else than a context.
