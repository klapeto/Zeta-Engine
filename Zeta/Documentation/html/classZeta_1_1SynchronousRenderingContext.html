Detailed Description

template<typename DisplayT>
class Zeta::SynchronousRenderingContext< DisplayT >

Template for creating Synchronous Rendering Contexts The Synchronous Contexts are the ones that run on the same thread they are created The template parameter is the Class that will create the Display Object
